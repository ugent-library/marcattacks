# alma2else

Turn Clarivate Alma XML exports into something else.

## Build

```
npm install
```

```
npm run build:ts
```

```
npm link
```

### Run

Generate JSON

```
alma2else --to json <file>
```

Generate Aleph sequential

```
alma2else --to alephseq <file>
```

Generate Prolog

```
alma2else --to prolog <file>
```
